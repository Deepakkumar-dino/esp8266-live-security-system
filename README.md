# esp8266-live-security-system![Visitors](https://komarev.com/ghpvc/?username=deepakkumar-sp&repo=esp8266-live-security-system)
ESP8266 Live Security System

Description:
A real-time security monitoring system using ESP8266, ultrasonic sensor, and sound sensor.
The system detects intrusion using multi-condition logic and displays live status via web server and ThingSpeak.

Hardware Used:
- ESP8266 (ESP-12E / NodeMCU)
- Ultrasonic Sensor (HC-SR04)
- Sound Sensor
- Breadboard, Jumper Wires

Features:
- Live distance monitoring
- Sound level detection
- Intrusion alert logic
- Web dashboard display
- Cloud logging (ThingSpeak)

Future Improvements:
- Buzzer alert
- Telegram notification
- Camera integration
